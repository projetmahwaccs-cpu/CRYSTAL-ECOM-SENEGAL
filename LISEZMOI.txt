TOUT UPLOADER A LA RACINE DU REPO GITHUB (a cote des images/videos) :

  FICHIERS PARTAGES (obligatoires, le site ne marche pas sans) :
    index.html        page principale (accueil, boutique, fiche produit,
                      panier, commande, confirmation, favoris, contact,
                      devenir vendeur)
    products.js       les 188 produits + categories (POUR AJOUTER UN
                      PRODUIT : c'est CE fichier qu'on modifie, une
                      seule fois pour tout le site)
    ccl-core.js       moteur des sous-pages (panier, cartes, filtres)
    ccl-styles.css    design partage de toutes les pages

  12 PAGES CATEGORIES :
    mode.html                beaute.html            alimentation.html
    maison-nouveautes.html   maison-deco.html       high-tech.html
    papeterie.html           sport.html             auto.html
    supermarche.html         bebe.html              promos.html

  3 PAGES PARTENAIRES :
    bestfoods.html    capital.html    ssr.html
    ATTENTION : capital.html et ssr.html REMPLACENT vos anciennes
    pages du meme nom. Si vous voulez garder les anciennes versions,
    renommez-les avant (ex: ssr_ancien.html) ou n'uploadez pas les
    nouvelles.

COMMENT CA MARCHE :
  - Le panier et les favoris sont synchronises sur TOUTES les pages
    (localStorage ccl_cart / ccl_wishlist)
  - Ajouter au panier marche depuis n'importe quelle page
  - Cliquer un produit ouvre sa fiche : index.html#/produit/ID
  - Panier / commande / paiement : tout passe par index.html
  - Chaque page a sa propre adresse indexable par Google :
    votre-site/mode.html, votre-site/bestfoods.html, etc.
